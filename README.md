# OpenSpec OPSX-SDD · 全域 Agent Skill 安裝指南

## 目錄 1.Github Copilot

- 把opsx-sdd目錄複製至 C:\Users\<用戶名>\.vscode\workspace\.github\skills\ 目錄下

- 把prompt目錄複製至 C:\Users\<用戶名>\.vscode\workspace\.github\ 目錄下


## 目錄 2.Antigravity

- 把opsx-sdd目錄複製至 C:\Users\<用戶名>\.gemini\antigravity\workspace\.agent\skills\ 目錄下

- 把workflows目錄複製至 C:\Users\<用戶名>\.gemini\antigravity\workspace\.agent\ 目錄下


## 目錄 3.Cursor

- 把opsx-sdd目錄複製至 C:\Users\<用戶名>\.cursor\skills\ 目錄下

- 把commands目錄複製至 C:\Users\<用戶名>\.cursor\workspace\.cursor\ 目錄下
